### 1. Push code to a GitHub repository
```bash

* Go to your repository on GitHub
* Click on **Settings > Pages**
* Under **Source**, select `main` branch and `/ (root)`
* GitHub will provide a live URL:
  `https://yourusername.github.io/portfolio/`
